# Public Case Request Form Help Content

This document defines all tooltips, inline help text, and guidance for Public Case Request Forms. Follow the standards defined in [TOOLTIP_INLINE_HELP_STANDARDS.md](./TOOLTIP_INLINE_HELP_STANDARDS.md).

## Form-Level Guidance (First-Time Users)

### Purpose Statement
> "This secure form submits a case request to the organization. Your submission creates a pending request that will be reviewed by staff before becoming an active case."

### Who Should Use This Form
> "Use this form if you are a client, attorney, claims adjuster, or authorized representative submitting a new case request. You do not need an account."

### What Happens After Submission
> "After submission, you'll receive a confirmation with your request reference number. Staff will review your request and may contact you for additional information. Once approved, your request becomes an active case."

### Security Notice
> "This form uses secure, encrypted transmission. Uploaded files are stored in compliance with industry standards. Only authorized organization staff can access your submission."

## Common Mistakes to Avoid

| Scenario | Warning Text | Location |
|----------|--------------|----------|
| Empty subject photo for surveillance | "Surveillance cases are more effective with a recent subject photo." | Subject photo field |
| Missing claim number for insurance | "Insurance cases require a claim number for proper routing." | Claim number field |
| Budget set to $0 | "A budget of $0 may delay case assignment." | Budget field |
| No services selected | "Selecting services helps staff assign the right investigators." | Services field |
| Incomplete address | "Complete addresses improve location success rates." | Address fields |

## Component Usage Reference

### Tooltips (hover-triggered)
```tsx
import { HelpTooltip } from "@/components/ui/tooltip";

<HelpTooltip content="Tooltip text here" />
```

### Inline Help Text
```tsx
<p className="text-xs text-muted-foreground mt-1">
  Inline help text here.
</p>
```

### First-Time User Tooltips
```tsx
import { DelayedTooltip } from "@/components/ui/tooltip";

<DelayedTooltip content="Critical field help" delayDuration={0}>
  <TriggerElement />
</DelayedTooltip>
```

### Warning Alerts
```tsx
import { Alert, AlertDescription } from "@/components/ui/alert";
import { AlertTriangle } from "lucide-react";

<Alert>
  <AlertTriangle className="h-4 w-4" />
  <AlertDescription>Warning message here.</AlertDescription>
</Alert>
```
